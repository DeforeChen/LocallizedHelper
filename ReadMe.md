# ReadMe

## 用途

基于 [CatchZeng/Localizable.strings2Excel](https://github.com/CatchZeng/Localizable.strings2Excel) 做的可视化封装。在 `MacOS` 下运行的桌面端 `App`

通过 `Excel` 文件生成移动端不同平台下所需的多语言文件（`iOS` 下是 `.strings`,  `Android` 下为 `.xmls` ）
